# Assets Directory

This directory should contain:

- `icon.png` - App icon (1024x1024 recommended)
- `splash.png` - Splash screen image
- `adaptive-icon.png` - Android adaptive icon (foreground)
- `favicon.png` - Web favicon

Place your app assets here according to the Expo asset requirements.

## Quick Setup

You can use the Expo asset generator or place your own images here.

Minimum requirements:
- **icon.png**: Square icon for the app (1024x1024 or larger)
- **splash.png**: Splash screen (recommended 1242x2436 for iPhone or similar)
- **adaptive-icon.png**: Foreground image for Android adaptive icon
- **favicon.png**: Web favicon (48x48 or larger)
